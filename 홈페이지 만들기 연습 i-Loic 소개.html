i-Logic(설계 자동화 구현 프로그래밍 언어)

    Inventor 는 자동화 설계 구현 API (Application Programming Interface) 인 "i-Logic" 을 기본적으로 제공합니다.

    * iLogic 을 통해 다양한 산업분야 및 제조 아이템에 적용하여 설계자동화를 구현할 수 있습니다.>
